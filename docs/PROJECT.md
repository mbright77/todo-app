# Todo App

## Product
A local-only, offline-first todo app for mobile and desktop.

No accounts.
No server.
No cloud sync.

## Users
Single user on one device.

## Core Features
- Create tasks
- Complete tasks
- Edit tasks
- Delete tasks
- Task lists (Active, Upcoming, Completed)
- Search

## Data Rules
All data is stored locally on the device.
No remote APIs.
No telemetry.
No analytics.

## Technology
Frontend: React + TypeScript
Platform: PWA
Storage: IndexedDB (or SQLite via WASM)

## Non-Goals
- No user accounts
- No sync
- No sharing
- No backend
